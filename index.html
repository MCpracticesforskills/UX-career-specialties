Consider These 3 New UX Speciality Careers before Going to College

It's no secret colleges aren't what they used to be. Colleges are outdated, irrelevent, and
outrageously expensive. Instead, consider a career that pays well and doesn't require a college degree!

Explore these Careers:

UX Researcher
UX Writer
UX/UI Developer

UX Researcher

Salary Range:$69,000-$130,000 according to PayScale

Work From Home: Yes

Summary: A UX Researcher is responsible for researching user trends, conducting user testing, and creating user-centered designs. A good UX Researcher can tell stakeholders what their users want from their product/service.

Here is a typical job description for a UX Researcher:
  • Have a hands-on approach with the process of research, prototyping, development, and testing
  • Ability to communicate with users, stakeholders and developers
  • Has knowledge of mobile design, psychology, interaction design, graphic design, and marketing disciplines
  • Maintain and perform functional and visual updates on company products
  • Create wireframes, mockups, prototypes, and storyboards
  • Create user journey maps, user flows, use cases and user stories

UX Writer

Salary Range:$67,500-$161,000 according to ZipRecruiter

Work From Home: Yes

Summary: A UX Writer is responsible for researching the best strategy for deploying content. They typically write for APIs. A good UX Writer can discern between copywriting and instructional writing; they know what the user wants to hear.

Here is a typical job description for a UX Writer according to 4CornerResources:
  • Conduct competitive analyses of other websites that operate in the same space
  • Prepare customer analysis based on demographic research and user transactions with API
  • Create a content development strategy
  • Coordinate with UI design teams on issues like navigation, page routing, product page design, etc.
  • Track usability goals and prepare reports for upper management
  • Conduct usability testing then create a report showcasing the findings

UX/UI Developer

Salary Range:$53,000-$152,000 according to ZipRecruiter

Work From Home: Yes

Summary: A UI/UX Developer, sometimes refered to as a front-end web developer, is responsible for applying interactive and aesthetic principles on websites and web applications for a positive, cohesive user experience. A Developer must be highly skilled in HTML, CSS, JavaScript, and other programming languages.

Here is a typical job description for a UX/UI Developer according to 4CornerResources:
  • Conduct testing of completed apps, websites, and software to assess user experience
  • Use code to develop visual aesthetics within a website, app, or product
  • Code websites, apps from start to finish, depending on what type of company one works for
  • Enhance user experience by creating seamless navigation through various digital interfaces
  • Develop ease-of-use concepts for improving user experience ensuring all interactions are convenient for customers
  • Collaborate with back-end web developers and programmers to improve usability
  • Present your designs to key stakholders, clients, colleagues, etc.
